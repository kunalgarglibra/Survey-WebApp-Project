User Features

LOGIN

In order to answer some surveys you would first need to be logged in. Log in by simply going to the login page from the dropdown or click here.

TAKE SURVEY

Select your survey published by the admin and select your desired answer for each question.

SUBMIT

After you're done with answering all the questions of the survey, just simply submit it using the submit button at the bottom of the survey.

Admin Features

MANAGE USERS

Only the admin has control for adding users and giving them their credentials to submit a survey that the admin has created for them. Moreover, the admin can also edit user details and delete them if needed.

MANAGE SURVEYS

Only the admin has control for adding surveys for users and publish them to all the users at the same time. Moreover, the admin can also edit the survey by altering the questions and/or their options. Once the survey gets published to the users, admin loses the control to edit the surveys.

VIEW USER RESULTS

Once the survey has been submitted by the user, all the survey answers can be then viewed by the admin. The admin can then analyze each user's response effectively using our View Results tab.
